# Neuraloft AI — AI Automation Agency Website

A production-grade, single-page website for an AI automation agency.
Built with **React 19 + TypeScript + Vite + Tailwind CSS v4**, fully
responsive (mobile / tablet / desktop) and component-organized.

---

## Quick start

```bash
npm install      # install dependencies
npm run dev      # start dev server  → http://localhost:5173
npm run build    # production build  → dist/
npm run preview  # preview the built site
npm run typecheck# run TypeScript type-checking
```

> `node_modules/` and `dist/` are **not** included in this zip — they are
> generated by `npm install` and `npm run build`. `package-lock.json` **is**
> included so installs are reproducible.

Requires **Node.js 20.19+ or 22.12+**.

## Customizing

Almost everything you'd want to edit lives in **`src/data/content.ts`** —
brand name, services, pricing plans, FAQ, testimonials, dashboard data and
the chat demo's canned replies. Change copy there without touching components.

**Theme colors & fonts** are defined as tokens in `src/index.css` under the
`@theme` block (`--color-mint`, `--color-indigo`, fonts, etc.). Tailwind v4
auto-generates utilities (`bg-mint`, `text-indigo`, `font-display`…).

---

## Before you publish — important

This is a **demo brand**. Replace the following with your real business:

- **Brand name / email** — `src/data/content.ts` (`brand` object) + page `<title>` in `index.html`.
- **Result metrics & testimonials** — the figures in `resultMetrics` and the
  `testimonials` array are clearly-marked **placeholders**. Use only real,
  verified numbers and quotes you have permission to publish.
- **Pricing** — confirm the plan prices match your real offer.
- **Dashboard / chat demo** — these are front-end UI demos (no backend).
  Wire them to real data/AI APIs before presenting them as live products.
- **Contact form** — currently front-end only (shows a confirmation toast).
  Connect it to an email service, CRM, or embed your Calendly link.
- **Social links** in the footer point to `#` — add your real URLs.

## Tech notes

- **No three.js** — the animated background uses a lightweight 2D canvas
  particle network instead, so it stays smooth on phones.
- `prefers-reduced-motion` is respected (heavy animations disabled).
- The bundle includes `recharts`; if you want a smaller build you can lazy-load
  the `Dashboard` section with `React.lazy`.

Built as a clean, multi-file refactor — every component in its own file.
